Class Curve

Curve.schedule (self, schedule) schedule using the function schedule
Curve.cp_at (self, t) return the Control Point at t seconds
Curve.cp_before (self, t, create) return the last Control Point before t seconds
may create a Control Point if create is true
Curve.add_cp (self, t, new_cp) add a Control Point at t seconds
new_cp is a table containing val (reference value for next CP) and eval (returns value for input in range 0-1)
Curve.eval (self, t) evaluate the Curve at t seconds
Curve.update (self, dt) update the Curve by dt seconds
Curve.debug_draw (self, max, segments, width, height) debug-draw the Curve on screen (love2d only)
